MyMeds

A comprehensive mobile medication management system built with Flutter & Firebase

Project Overview

MyMeds is a comprehensive medication management system designed to help users track their prescriptions, set reminders, and monitor their adherence to medical treatments. It is a mobile application built using Flutter for the frontend and Firebase for the backend, providing a seamless experience for users to manage their health effectively.

Technologies Used

Flutter Dart Firebase

Project Context

This project was developed as a final year capstone project to address the common issue of medication non-adherence among elderly patients. The goal was to create a simple, intuitive mobile interface that could be used by patients and monitored by caregivers.

Key Features

  • Smart Medication Reminders: Push notifications that persist until medication is marked as taken.
  • Caregiver Dashboard: Allows family members to monitor adherence remotely.
  • User-friendly Interface: Large text and high-contrast buttons for accessibility.

My Contributions

  • Firebase Establishment: Setting up Firebase Authentication and Firestore for secure data management.
  • Push Notifications: Implementing persistent push notifications using Firebase Cloud Messaging.
  • UI/UX Design: Designing an accessible and intuitive user interface using Flutter's widget system.
  • Testing & Deployment: Conducting user testing and deploying the app.
